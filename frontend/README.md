# TinyURL Clone - Frontend

UI built with **React** and **Vite**. Designed to communicate with the TinyURL Clone backend.

## Technologies

- **Framework:** React 18
- **Build Tool:** Vite 
- **Styles:** Native CSS3 + React Icons
- **HTTP Client:** Axios
- **Environment:** Dockerized with Node.js Alpine

## Installation and Running

### Option A: With Docker (Recommended)
The frontend starts automatically along with the entire stack from the root of the project.

```bash
# From the root folder of the project
docker compose up --build
